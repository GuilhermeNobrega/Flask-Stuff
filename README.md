
## Instalação do Flask

Para começar a trabalhar com Flask, siga os passos abaixo para configurar seu ambiente local.

### Pré-requisitos

Certifique-se de ter Python e pip instalados. Você pode verificar instalando-os através do terminal:

```bash
python --version
pip --version
pip install Flask
flask --version
pip show flask
```
